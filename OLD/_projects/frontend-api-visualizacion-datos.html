Descripción

La empresa Cink contactó con nosotros para colaborar en el desarrollo de un proyecto para una compañía de banca. Este proyecto, consistía en el desarrollo de una solución para la medición y visualización de datos relativos a proyectos llevados a cabo por dicha compañía en distintos países del mundo.

Desde Delirium Coder nos hemos encargado del desarrollo de la parte frontend, así como de la API web que interconecta el frontend con la base de datos en la que el cliente recopila una serie de datos de impacto generados por la compañía.
